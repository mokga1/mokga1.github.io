전사 (The Warrior) 육성 공략

모든 전투의 선봉 — 창을 제외한 전 근접무기의 달인

캐릭터 평가기 구슬 시스템 초보 가이드 리세마라 전사 검투사 도둑 마법사 성직자 💬 피드백

개요

전사는 강인한 체력을 바탕으로 최전선에서 선제 공격을 담당하는 근접 딜러입니다. 창을 제외한 모든 근접무기(검·도끼·둔기)를 다룰 수 있어 무기 선택의 폭이 넓고, 회피술로 생존력도 챙길 수 있습니다.

초기 파라메터와 배분

STRAGRDEXINTVITMEN보너스
1261066626

공식 가이드 권장은 STR과 VIT를 15로 맞추는 것입니다. STR은 공격력(WC)과 장비 무게, VIT는 HP·SP 회복과 질병 저항에 직결됩니다. 남는 보너스는 명중(HC)의 DEX나 회피(DC)의 AGR에 투자하세요. 생성 시 15까지는 포인트 1로 올라가지만 이후엔 구슬 3~5개 구간이라, 15 세팅은 생성 화면에서 끝내는 게 정석입니다.

무기 기능 셋(검술·부술·둔기술) 중 하나만 골라 집중하는 것이 구슬 효율의 핵심입니다. 나눠 올리면 어떤 상위 기술도 해금하지 못한 채 구슬만 마릅니다.

전사의 핵심 유틸인 리뉴얼은 INT 15, 종결기인 실드블럭은 INT 25를 요구합니다. 순수 근접이라도 INT를 완전히 버리면 안 되는 이유입니다. 실드블럭은 방패 착용 시 최대 40%까지 피해를 줄여줍니다.

육성 로드맵

리세마라 구간 (Lv 1~50)

  • 목표: 주무기 기능 + 회피술 25/25. 구슬이 남지 않아도 달성하면 성공입니다.
  • 파라메터는 생성 시 배분(STR·VIT 15)으로 충분 — 이 구간에 구슬을 스탯에 쓰지 마세요.
  • 기능 21부터 성공률 10%의 도박 구간입니다. 실패가 누적되면 평가기의 리세마라 진단으로 손절 타이밍을 확인하세요.

본육성 구간 (Lv 50+)

  • 회피술 20을 채워 타이탄차지 확보 → 무기술 30으로 스위핑블로우 해금.
  • INT를 15(리뉴얼) → 25(실드블럭) 순으로 올립니다. INT 16부터는 구슬 3~4개 구간이니 계획적으로.
  • 남는 구슬은 STR 증강(공격력)이나 보조직업 기능에 투자합니다.
